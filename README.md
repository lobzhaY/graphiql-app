***Tech Stack***

- qraphql
- React-router-dom
- TypeScript
- Eslint
- Prettier
- react-toastify
- SCSS
- firebase
- react-hook-form
- react-i18next
- vite

_Deploy_
- gh-pages 
- youtube - video

***Installation***
--------------------------
1. Clone repository
```
$ git clone https://github.com/lobzhaY/graphiql-app.git
```
2. Install dependencies
```
$ npm i
```
3. Start
```
$ npm run dev
```
